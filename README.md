## 🔍 Frequently Asked Questions

**Does this software modify the game files permanently?**
No. The mod suite uses a virtual file system. When you launch the game through the suite, your mods apply. If you launch the game normally through your launcher, it runs the original, unmodified version.

**What happens if I update EA Sports FC 26?**
Updates might break existing mods. If your game stops working after an official update, open the suite and click the "Refresh" button. This forces the suite to re-verify game files and check for compatible mod versions.

**Can I use this online?**
Be careful with online play. EA Sports often blocks players who use mods in online modes. Rely on this suite for offline career mode or local kick-off matches to avoid issues with game service policies.

**The game does not launch. What do I do?**
Check your antivirus settings. Sometimes security software flags the mod loader as suspicious. Add the folder containing the suite to your antivirus exclusion list. Also, check that you have the latest .NET Desktop Runtime installed from the official Microsoft website.
